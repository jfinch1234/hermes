# Pilot Weekly Agenda

This agenda is fixed and non-negotiable for all Hermes pilots.

## Weekly agenda (35 minutes)
- 5 min: system health (CI, invariants, incidents)
- 10 min: trust metrics review (from TRUST_METRICS.md)
- 10 min: pilot usage patterns (not outcomes)
- 10 min: feedback triage (accepted / rejected)
- 10 min: next-week focus

## Explicitly excluded
- conversion rates
- sales metrics
- optimization requests
- UI "improvements"
