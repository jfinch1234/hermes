# Post-Pilot Review Memo

This memo is required to close a pilot. It must be written and reviewed before any conversion decision.

## Required structure
- Pilot summary
- Success signals observed
- Trust risks encountered
- Policy adherence confirmation
- Recommendation: Convert / Extend / Stop

## Required references
- Explicit reference to TRUST_METRICS.md
- Explicit reference to PILOT_SUCCESS_SIGNALS.md

## Requirements
- Written justification is mandatory.
- Conclusions must be evidence-based.

## Prohibited content
- ROI claims
- Anecdotal-only conclusions
