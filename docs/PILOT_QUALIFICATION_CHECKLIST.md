# Pilot Qualification Checklist

All items are required before any pilot begins.

## Required acceptance
- Accepts non-ranking and non-recommendation behavior
- Accepts max-3 options cap
- Accepts single-store context
- Accepts proof/audit visibility in dev/staging
- Confirms no paid placement requests
- Willing to sign pilot contract as-is

## Immediate disqualifiers
- Requests ranking, recommendations, or "best" labels
- Requests more than 3 options or multi-store comparison
- Requests paid placement, sponsorship, or promotional badges
- Requests conversion optimization or A/B testing
- Refuses pilot contract terms
