# Pilot Scope Definition

Pilot scope is locked before any code changes or demos.

## Fixed scope rules
- Single store only.
- Fixed catalog adapter.
- Fixed use case (one decision flow).
- Explicitly excluded features:
  - ranking or recommendations
  - additional stores or cross-store comparisons
  - personalization or paid placement
  - conversion optimization

## No expansion during pilot
Scope expansion is not permitted once the pilot begins.

## Scope Freeze statement
"We acknowledge that pilot scope is frozen to a single store, a fixed adapter, and one decision flow. We will not request scope expansion or additional features during the pilot."
